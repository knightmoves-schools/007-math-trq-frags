# 007 Math

[![007 Math](https://img.youtube.com/vi/r2Iv7Kogt_s/0.jpg)](https://www.youtube.com/watch?v=r2Iv7Kogt_s)

the Math.cs file
- should capture the result of 1 plus 4 in a class-level variable named `Sum`
- should capture the result of 5 minus 2 in a class-level variable named `Difference`
- should capture the result of 3 multiplied by 5 in a class-level variable named `Product`
- should capture the result of 15 divided by 2 in a class-level variable named `Quotient`
- should capture the remainder of 7 remainder of 2 in a class-level variable named `Remainder`
